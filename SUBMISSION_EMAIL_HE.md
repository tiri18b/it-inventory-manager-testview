# נוסח מייל לשליחת משימת הבית

היי מיכל,

מצורפת משימת הבית שביצעתי עבור Testview.

במסגרת המשימה בניתי כלי Desktop פשוט לניהול ציוד מחשוב בחברה, IT Inventory Manager.

המערכת נבנתה ב־C# עם Windows Forms, וכוללת הוספת ציוד, עריכה, מחיקה, חיפוש, סינון, שיוך ציוד לעובד, סטטוס ציוד ושמירת נתונים מקומית.

צירפתי גם README עם הוראות הרצה, קובץ AI_USAGE שמסביר איך נעזרתי בכלי AI בתהליך העבודה, וקישור לסרטון קצר שבו אני מציג את הפתרון ומה הייתי מוסיף בהמשך.
